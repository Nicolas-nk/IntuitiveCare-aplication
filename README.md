# Relação Operadoras Ativas ANS

Aplicação CRUD para IntuitiveCare

## Começando
```
cd client
npm install

```

```
cd server
npm install

```

### Para Iniciar digite no terminal dentro do repositório:
```
cd client
npm run serve
```
### E em outro terminal digite:
```
cd serve
npm run dev
```
